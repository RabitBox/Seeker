Title : Seeker

-ルール-
フィールドの奥にある「落し物」を見つけてください。

-操作方法-
矢印キーで左右に移動。同じ方向に素早く二回入力するとダッシュします。

-入力設定-
移動　　　：矢印キー
アクション：Aキー
ポーズ　　：Sキー

海外のブラウザゲームでよくある「クリックしか出来ない」や「十字キーの操作しか出来ない」
といった「プレイヤーに出来る行動が一つしかないゲーム」というのをテーマに作りました。
今出来る事を実践する為のプロジェクトの為、物語性は何もありません。
